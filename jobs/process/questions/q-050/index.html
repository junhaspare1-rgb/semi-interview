포토 공정의 불량은 크게 결함, CD 불량, 오버레이 불량으로 나눌 수 있습니다. 결함 종류는 첫째, 파티클로 코팅 장비, 노광 장비, 클린룸 환경에서 유입된 이물질이 PR 또는 마스크에 부착되어 패턴 결함을 일으킵니다. 둘째, 브릿지로 인접 라인이 연결되는 불량이며 현상 불량이나 CD가 너무 클 때 발생합니다. 셋째, 오픈은 라인이 단절되는 불량으로 CD가 너무 작거나 PR 접착 불량 시 발생합니다.

넷째, 핀홀은 PR 막에 미세한 구멍이 생기는 불량입니다. CD 불량은 노광 에너지, 현상 조건 불량, DOF 부족 등으로 발생하며, 오버레이 불량은 장비 정렬 오차, 웨이퍼 변형 등으로 발생합니다. 불량 검출은 광학 검사 장비를 이용한 인라인 검사로 진행하며, 검출된 결함의 위치와 패턴을 분석해 원인을 역추적합니다. 공정 관점에서는 각 장비의 PM 주기, 클린룸 청정도 관리, 공정 조건 SPC 모니터링이 예방의 핵심입니다.
