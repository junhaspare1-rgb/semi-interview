OPC(Optical Proximity Correction)란 무엇이며 왜 필요합니까?

40초 Script

OPC는 광근접 보정의 약자로, 광학적 회절 현상으로 인해 마스크 패턴과 실제 웨이퍼 패턴이 다르게 형성되는 문제를 보정하기 위해 마스크 패턴을 미리 변형하는 기술입니다. 빛이 마스크를 통과할 때 회절과 간섭 효과로 인해 패턴 가장자리가 둥글어지거나 코너가 안쪽으로 당겨지는 현상이 발생합니다. 특히 패턴이 미세해질수록 이러한 근접 효과가 심해져 목표 CD와 실제 CD의 차이가 커집니다.

모범 답안

OPC는 광근접 보정의 약자로, 광학적 회절 현상으로 인해 마스크 패턴과 실제 웨이퍼 패턴이 다르게 형성되는 문제를 보정하기 위해 마스크 패턴을 미리 변형하는 기술입니다. 빛이 마스크를 통과할 때 회절과 간섭 효과로 인해 패턴 가장자리가 둥글어지거나 코너가 안쪽으로 당겨지는 현상이 발생합니다. 특히 패턴이 미세해질수록 이러한 근접 효과가 심해져 목표 CD와 실제 CD의 차이가 커집니다.

OPC는 이를 보정하기 위해 마스크에 해머헤드 같은 작은 보조 도형을 추가하거나 패턴 가장자리를 미리 움직여서, 실제 웨이퍼에서는 원하는 패턴이 나오도록 합니다. OPC 처리는 매우 복잡한 광학 시뮬레이션이 필요하며, 첨단 공정에서는 마스크 파일이 매우 커지고 OPC 계산 시간이 수십 시간에서 수백 시간이 소요됩니다. 모델 기반 OPC와 규칙 기반 OPC가 있으며, 최근에는 AI, 머신러닝 기반의 OPC 계산 가속도 연구되고 있습니다.
