40초 Script

BARC는 포토레지스트 하부에 도포하는 반사 방지막으로, 기판에서 반사된 빛이 PR과 간섭을 일으켜 패턴 품질을 저하시키는 문제를 방지합니다. 노광 시 입사광이 PR을 통과해 기판 또는 하부 막에서 반사되면, 반사광이 입사광과 간섭해 정재파가 생기고, 이로 인해 PR 내부에 주기적인 명암 패턴이 형성되어 패턴 측벽이 물결 모양이 되는 현상이 발생합니다. BARC는 반사율을 수 % 이하로 낮춰 이 문제를 해결합니다.

모범 답안

BARC는 포토레지스트 하부에 도포하는 반사 방지막으로, 기판에서 반사된 빛이 PR과 간섭을 일으켜 패턴 품질을 저하시키는 문제를 방지합니다. 노광 시 입사광이 PR을 통과해 기판 또는 하부 막에서 반사되면, 반사광이 입사광과 간섭해 정재파가 생기고, 이로 인해 PR 내부에 주기적인 명암 패턴이 형성되어 패턴 측벽이 물결 모양이 되는 현상이 발생합니다.

BARC는 반사율을 수 % 이하로 낮춰 이 문제를 해결합니다. BARC의 광학 설계 원리는 박막 간섭 이론으로, 막 두께와 굴절률을 조절해 PR과 기판 사이에서 반사된 빛과 BARC 표면에서 반사된 빛이 상쇄 간섭을 일으키도록 합니다. 소재는 유기계와 무기계로 나뉩니다. 유기 BARC는 스핀 코팅 방식으로 도포하며 식각 시 PR과 함께 제거되고, 무기 BARC는 CVD로 증착하며 식각 하드 마스크 역할을 겸하기도 합니다.

TARC라 불리는 상부 반사 방지막도 있습니다.
