MOSFET 문턱 전압(Threshold Voltage)이란 무엇이며, 어떻게 결정되는지 설명해주세요.

Vt강반전게이트 산화막 두께기판 도핑일함수
답변 예시

문턱 전압은 MOSFET에서 채널이 형성되어 전류가 흐르기 시작하는 최소 게이트 전압입니다. NMOS에서는 게이트 전압이 올라가면 기판 표면의 정공이 밀려나고 전자가 모여 강반전 상태가 됩니다. 문턱 전압은 게이트 산화막 두께, 기판 도핑 농도, 게이트와 기판의 일함수 차이, 산화막 전하에 의해 결정되며 공정에서는 Vt 임플란트로 조정합니다.

Vt가 너무 낮으면 누설이 커지고, 너무 높으면 구동 전류가 줄기 때문에 성능과 전력의 균형이 중요합니다.

DIBL펀치스루Halo 임플란트LDDHigh-k
답변 예시

단채널 효과는 MOSFET 채널 길이가 짧아지면서 게이트가 채널을 충분히 제어하지 못하는 현상입니다. 대표적으로 문턱전압 감소, DIBL, 펀치스루, 누설전류 증가가 나타납니다. 해결 방법은 Halo 임플란트와 LDD로 전기장을 완화하고, High-k 절연막으로 게이트 제어력을 높이며, FinFET이나 GAA처럼 게이트가 채널을 더 많이 감싸는 구조를 쓰는 것입니다.

결국 미세화에서는 채널 길이를 줄이면서도 게이트 지배력을 유지하는 구조와 도핑 설계가 핵심입니다.

해상도(Resolution)와 DOF(Depth of Focus)란 무엇이며, 서로 어떤 관계입니까?

레일리 공식k1NAλDOF 트레이드오프
답변 예시

Resolution은 Lithography에서 가까운 두 Pattern을 구분해 원하는 CD로 전사할 수 있는 능력입니다. 파장 λ가 짧고 NA가 클수록 Resolution은 좋아집니다. 반면 DOF, 즉 Depth Of Focus는 초점이 일부 벗어나도 Pattern 품질이 유지되는 초점 Margin입니다. DOF는 NA가 커질수록 감소하므로, Resolution을 높이기 위해 NA를 키우면 DOF가 줄어드는 Trade-Off가 생깁니다.

따라서 미세공정에서는 해상도 확보와 초점 Margin 유지의 균형이 중요합니다.

OPC는 광근접 보정의 약자로, 포토 공정에서 마스크 패턴이 웨이퍼에 왜곡되어 전사되는 현상을 보정하기 위해, 마스크 형상을 미리 수정하는 기술입니다. 미세 패턴에서는 빛의 회절과 간섭, 인접 패턴 영향으로 선폭이 변하거나 모서리가 둥글어지고 CD 오차가 발생합니다. 이를 줄이기 위해 마스크 패턴의 폭, 위치, 모서리를 조정하거나 보조 패턴을 넣습니다.

결국 OPC는 최종 웨이퍼 패턴을 설계값에 가깝게 만들고, CD 균일도와 수율을 확보하기 위해 필요합니다.

ArF보다 파장이 짧아 다중 Patterning을 줄이고 해상도를 높일 수 있지만, Mask 결함, PR 감도, Stochastic Defect, 장비 비용 관리가 어렵습니다.

LELESADPSAQP스페이서피치 1/2·1/4
답변 예시

멀티패터닝은 한 번의 노광으로 만들기 어려운 미세 패턴을 여러 번의 포토와 식각 공정으로 나누어 형성하는 기술입니다. 대표적으로 LELE, SADP, SAQP가 있습니다. LELE는 패턴을 두 번 나누어 노광하고 식각하는 방식이고, SADP는 Mandrel 옆에 Spacer를 형성한 뒤 Mandrel을 제거해 Pitch를 줄이는 방식입니다.

SAQP는 Spacer 공정을 반복해 더 미세한 Pitch를 구현합니다. 장점은 ArF 장비로 미세화를 이어갈 수 있다는 점이고, 단점은 공정 수 증가, Overlay 오차, CD 균일도 관리가 어렵다는 점입니다.

답변 예시

포토 공정의 불량은 크게 결함, CD 불량, 오버레이 불량으로 나눌 수 있습니다. 결함 종류는 첫째, 파티클로 코팅 장비, 노광 장비, 클린룸 환경에서 유입된 이물질이 PR 또는 마스크에 부착되어 패턴 결함을 일으킵니다. 둘째, 브릿지로 인접 라인이 연결되는 불량이며 현상 불량이나 CD가 너무 클 때 발생합니다. 셋째, 오픈은 라인이 단절되는 불량으로 CD가 너무 작거나 PR 접착 불량 시 발생합니다.

넷째, 핀홀은 PR 막에 미세한 구멍이 생기는 불량입니다. CD 불량은 노광 에너지, 현상 조건 불량, DOF 부족 등으로 발생하며, 오버레이 불량은 장비 정렬 오차, 웨이퍼 변형 등으로 발생합니다. 불량 검출은 광학 검사 장비를 이용한 인라인 검사로 진행하며, 검출된 결함의 위치와 패턴을 분석해 원인을 역추적합니다. 공정 관점에서는 각 장비의 PM 주기, 클린룸 청정도 관리, 공정 조건 SPC 모니터링이 예방의 핵심입니다.
